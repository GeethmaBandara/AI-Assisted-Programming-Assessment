\# AI-Assisted Programming Assessment



\## Student Result Management System



This project was developed for the Advanced Programming Concepts assessment on AI-Assisted Programming.



The purpose of the project is to compare traditional programming and AI-assisted programming approaches using a Java console-based Student Result Management System.



\## Project Versions



\### Version A - Traditional Programming



Version A was developed using a traditional programming approach.



The system supports:



\- Adding students

\- Entering student marks

\- Calculating average marks

\- Determining student grades

\- Viewing student results

\- Searching for students

\- Displaying all student records



During testing, several weaknesses were identified:



\- Marks greater than 100 were accepted.

\- Negative marks were accepted.

\- Non-numeric input caused an InputMismatchException.

\- Duplicate student IDs were accepted.



\### Version B - AI-Assisted Programming



Version B was developed with assistance from ChatGPT.



AI assistance was used for:



\- System design improvement

\- Input validation

\- Exception handling

\- Duplicate student ID prevention

\- Code refactoring

\- Improving readability and maintainability



The AI-assisted version introduces an InputValidator class and improved separation of responsibilities.
